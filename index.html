lambdacube

                        
→

                    
                    
                        My Dissertation Project, a 3d tool to help explain the
                        concept of Henk Barendregt's 'lambdacube', written in
                        elm.
                    
                    
                        The lambda cube is a way of demonstrating how any
                        combination of three different type systems can be added
                        to the simply typed lambda calculus, and how the system
                        changes based on type system.
